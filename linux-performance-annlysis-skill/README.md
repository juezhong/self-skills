# linux-performance-analysis skill
## 功能描述
诊断 Linux 系统在 CPU、内存、I/O 和网络方面的性能问题与瓶颈。适用于用户要求分析出现以下状况的 Linux 主机：运行缓慢、过载、卡顿、超时、丢包、内存交换、磁盘阻塞，或显示高负载 / 高延迟。目标是定位瓶颈所在的子系统、判断瓶颈出现在内核态还是应用态，并定位应用层中导致问题的具体进程。
